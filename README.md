# API de Pesquisa de Satisfação

Projeto de estudo sobre a linguagem C#.

# Funcionalidades!

  - CRUD Tipos de pergunta ao usuário.
  - CRUD Tipos de Respostas/Pontuação.
  - Envio das respostas.

# Tecnologias Utilizadas
  - Banco MySQL 8.0.21.
  - EntityFramework Core 3.1.7.
  - Swagger 5.5.1
